## 題目 5：受詛咒的直線 (The Cursed Line) - Bonus +20分

### 故事背景

河內塔的傳說有了新的篇章。大梵天的三根金剛石柱 (1, 2, 3) 排成一直線，但是柱子 1 和柱子 3 之間因為距離太遠，產生了空間裂縫，導致**圓盤無法直接在 1 和 3 之間傳送**。這意味著所有的移動都必須經過中間的柱子 2。也就是說，你只能執行 $1 \leftrightarrow 2$ 或 $2 \leftrightarrow 3$ 的移動，嚴禁 $1 \leftrightarrow 3$ 的直接移動。

### 題目說明

讀取一個整數 `n`，印出將 `n` 個圓盤從 **柱子 1** 移動到 **柱子 3** 的完整步驟。

**要求：**

1. **必須**實作並使用遞迴函數：
    - `void hanoi(int n, int from, int to, int aux)`
2. **限制：**
    - **嚴格禁止**直接將圓盤從 1 移動到 3 (或 3 到 1)。
    - **嚴格禁止使用任何迴圈**。

### 輸入格式

一行包含一個整數：`n` (1 ≤ n ≤ 6)

*注意：因為移動次數會隨 n 增長得比原本更快 (3^n - 1)，所以 n 的上限較小。*

### 輸出格式

每一步移動輸出一行，格式為：`Move disk [編號] from [來源柱] to [目標柱]`

### 範例

**Sample Input 1**

```jsx
1
```

**Sample Output 1**

```jsx
Move disk 1 from 1 to 2
Move disk 1 from 2 to 3
```

**Sample Input 2**

```jsx
2
```

**Sample Output 2**

```jsx
Move disk 1 from 1 to 2
Move disk 1 from 2 to 3
Move disk 2 from 1 to 2
Move disk 1 from 3 to 2
Move disk 1 from 2 to 1
Move disk 2 from 2 to 3
Move disk 1 from 1 to 2
Move disk 1 from 2 to 3
```
