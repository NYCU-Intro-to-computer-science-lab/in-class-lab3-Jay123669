## Problem 4: Footsteps of Recursion - 30 pts

### Story Background

Although the recursive definition of the Fibonacci sequence is elegant, how does a computer actually calculate it? To let everyone intuitively see the unfolding and backtracking process of the "recursion tree", let's create a beautiful visual output to see how the program "climbs the tree" step by step to find the answer!

### Problem Description

Read an integer `n`, calculate the `n`-th Fibonacci number $F(n)$.

Definitions (starting from 1):

- $F(1) = 1$
- $F(2) = 1$
- $F(n) = F(n-1) + F(n-2)$

**Requirements:**

1. **Must** implement and use the recursive function:
    - `int fib(...)`: Calculates the Fibonacci number and handles indented output (please design the necessary parameters yourself).
2. **Restrictions:** This problem **strictly prohibits the use of any loops** (for, while, do-while). Violations will result in no points. TAs will check keywords.
3. **Upon entering the function**: First print indentation, then print `SEARCH fib(n)`.
4. **Before leaving the function**: First print indentation, then print `GET fib(n) = [Result]`.
5. **Indentation Rules**: Print the corresponding number of `|--` based on the current recursion depth. For example, depth 1 prints 1 `|--`, depth 2 prints 2 `|--`, and so on. (Depth 0 is the outermost layer, no `|--` needed).
6. Be sure to **calculate F(n-1) first, then calculate F(n-2)**.

### Input Format

A single line containing an integer: `n` (1 ≤ n ≤ 10)

### Output Format

- Process: Print the process of recursive calls and returns as shown in the example.
- Last line: Print the final calculation result.

### Example

**Sample Input 1**

```jsx
3
```

**Sample Output 1**

```jsx
SEARCH fib(3)
|--SEARCH fib(2)
|--GET fib(2) = 1
|--SEARCH fib(1)
|--GET fib(1) = 1
GET fib(3) = 2
2
```

**Sample Input 2**

```jsx
4
```

**Sample Output 2**

```jsx
SEARCH fib(4)
|--SEARCH fib(3)
|--|--SEARCH fib(2)
|--|--GET fib(2) = 1
|--|--SEARCH fib(1)
|--|--GET fib(1) = 1
|--GET fib(3) = 2
|--SEARCH fib(2)
|--GET fib(2) = 1
GET fib(4) = 3
3
```
